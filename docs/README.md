# docs/

Documentation hub for the project.

- `quickstart.md` – copy-paste-ready commands to get up and running.
- `structure.md` – explains the project layout and how components fit together.
- `er_diagram.md` – Entity-Relationship diagram and normalization notes (3NF).
- `sql.md` – summarizes schema design and example analytic queries; notes on indexes and performance.
- `api.md` – API endpoints and examples.
- `teacher_goals_mapping.md` – mapping of this scaffold to your teacher's goals; extend as needed.
